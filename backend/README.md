# Veda Cards Backend

Серверная часть приложения Veda Cards. Проект реализован на языке Kotlin с использованием фреймворка Spring Boot. Сервис предоставляет RESTful API для взаимодействия с мобильными клиентами, а также включает веб-интерфейс администратора для управления контентом.

## Архитектура проекта

Приложение следует модульной архитектуре и разделено на следующие функциональные компоненты:

* **`:app`** — Точка входа. Содержит конфигурацию сборки и настройки Spring Boot приложения.
* **`:common`** — Общий модуль. Включает утилитарные классы, модели ошибок и глобальные обработчики исключений.
* **`:feature:identity`** — Модуль безопасности. Отвечает за аутентификацию, управление пользователями и сессиями.
* **`:feature:cards`** — Доменная логика. Управление карточками, категориями и контент-блоками.
* **`:feature:admin`** — Модуль административной панели.

## Справочник API

Ниже представлен список доступных эндпоинтов REST API.

### Управление учетными записями и сессиями

| Метод | Эндпоинт | Описание |
| :--- | :--- | :--- |
| **POST** | `/api/accounts` | Регистрация нового пользователя |
| **GET** | `/api/accounts/me` | Получение профиля текущего пользователя |
| **POST** | `/api/accounts/check-email` | Проверка доступности email адреса |
| **POST** | `/api/sessions` | Аутентификация (вход в систему) |
| **POST** | `/api/sessions/refresh` | Обновление токена доступа (Refresh Token) |
| **DELETE** | `/api/sessions/logout` | Завершение сессии (выход из системы) |

### Карточки и контент

| Метод | Эндпоинт | Описание |
| :--- | :--- | :--- |
| **GET** | `/api/cards` | Получение списка карточек (поддерживает пагинацию) |
| **POST** | `/api/cards` | Создание новой карточки |
| **GET** | `/api/cards/{id}` | Получение данных карточки по идентификатору |
| **PUT** | `/api/cards/{id}` | Обновление данных карточки |
| **DELETE** | `/api/cards/{id}` | Удаление карточки |
| **GET** | `/api/cards/slug/{slug}` | Получение карточки по уникальному текстовому идентификатору (slug) |
| **GET** | `/api/cards/search` | Полнотекстовый поиск по карточкам |

### Категории

| Метод | Эндпоинт | Описание |
| :--- | :--- | :--- |
| **GET** | `/api/categories` | Получение списка всех категорий |
| **POST** | `/api/categories` | Создание новой категории |
| **GET** | `/api/categories/{id}` | Получение данных категории по идентификатору |
| **PUT** | `/api/categories/{id}` | Обновление категории |
| **DELETE** | `/api/categories/{id}` | Удаление категории |

### Системные

| Метод | Эндпоинт | Описание |
| :--- | :--- | :--- |
| **GET** | `/api/health` | Проверка состояния сервиса (Health Check) |
