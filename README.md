# Veda-cards

Благодарим вас за интерес к проекту Veda.cards

**Veda.cards** — это мобильное приложение-энциклопедия культуры и мифологии, в котором информация представлена в формате карточек. Пользователь собирает их в личную библиотеку, открывает новые знания о культурном наследии и постепенно изучает темы, отслеживая свой прогресс.

> [!IMPORTANT]
> **Это не игра**, а образовательный инструмент с карточной механикой навигации.

## Цели проекта

- **Образовательная**: Предоставить структурированные и достоверные знания о культуре, мифологии и традициях в удобном формате карточек, поддерживающем поэтапное изучение материала.
- **Сохранение и популяризация культурного наследия**: Сделать знания о мифах, традициях, обрядах и фольклоре разных народов доступными для широкой аудитории, способствуя их сохранению в цифровую эпоху.

## Структура репозитория

Проект организован как многорепозиторий, объединяющий клиентскую и серверную части:

```
.
├── mobile/          # Клиентская часть
│   ├── android/     # Android приложение
│   └── ios/         # IOS приложение
└── backend/         # Серверная часть
```

Детальная информация каждой части проекта находится в соответствующих директориях: `mobile/ios/README.md`, `mobile/android/README.md` и `backend/README.md`

## Технический стек

Клиентская часть включает приложения для операционных систем iOS и Android. Версия для iOS разработана на языке Swift с использованием SwiftUI, а для Android — на Kotlin с применением Jetpack Compose. Серверная часть (бэкенд) написана на Kotlin и Spring Boot; она предоставляет API для управления контентом и пользователями. Сборка проекта осуществляется с помощью Gradle (Kotlin DSL), а в качестве основного хранилища данных используется PostgreSQL.
